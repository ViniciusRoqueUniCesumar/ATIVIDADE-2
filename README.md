 CRUD de Biblioteca em Dart

 Descrição do sistema

Este sistema é um CRUD de biblioteca desenvolvido em Dart utilizando Programação Orientada a Objetos. Ele permite cadastrar, listar, atualizar e remover livros através de um menu interativo no terminal.


Classe Livro

Representa um livro dentro do sistema. Possui os atributos:

 id
titulo
 autor

Também possui um método para exibir os dados do livro.

Classe Biblioteca

Responsável por gerenciar os livros. Ela contém:

Uma lista de livros
 Métodos para:

   Criar livro
   Listar livros
   Atualizar livro
   Deletar livro



Como executar

1. Ter o Dart instalado na máquina
2. Abrir o projeto no VS Code
3. Rodar o seguinte comando no terminal:

dart main.dart



Exemplos de uso

Ao rodar o sistema, aparecerá um menu com opções:

 Criar livro
 Listar livros
 Atualizar livro
 Deletar livro
 Sair

Exemplo:

Criar livro: informar título e autor
 Listar: mostra todos os livros cadastrados
Atualizar: altera dados pelo ID
Deletar: remove um livro pelo ID
